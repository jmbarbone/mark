# Drop levels

Drop unused levels of a factor

## Usage

``` r
drop_levels(x, ...)

# S3 method for class 'data.frame'
drop_levels(x, ...)

# S3 method for class 'fact'
drop_levels(x, ...)

# S3 method for class 'factor'
drop_levels(x, ...)
```

## Arguments

- x:

  A `factor` or `data.frame`

- ...:

  Additional arguments passed to methods (not used)
