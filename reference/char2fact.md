# Character to factor

Converts characters to factors

## Usage

``` r
char2fact(x, n = 5)

# Default S3 method
char2fact(x, n = 5)

# S3 method for class 'character'
char2fact(x, n = 5)

# S3 method for class 'factor'
char2fact(x, n = 5)

# S3 method for class 'data.frame'
char2fact(x, n = 5)
```

## Arguments

- x:

  A vector of characters

- n:

  The limit to the number of unique values for the factor
