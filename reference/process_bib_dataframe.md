# Process bib values

Generates a data frame of values from bibs

## Usage

``` r
process_bib_dataframe(categories, values, fields, keys)
```

## Arguments

- categories:

  A list of categories

- values:

  A list of values

- fields:

  a Vector of fields

- keys:

  a Vector of keys

## Value

A wide `data.frame` with explicit `NA`s
